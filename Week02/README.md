# Laporan Praktikum Minggu 2

Kumpulan program Dart untuk pembelajaran pemrograman mobile.

## Daftar Program

### I. TUGAS INDIVIDUAL

1. **BMI Calculator** (`Body_Mass_Index.dart`)   - Menghitung Body Mass Index berdasarkan berat dan tinggi
   - Menampilkan kategori BMI (Kurus, Normal, Gemuk, Obesitas tingkat 1, dan Obesitas tingkat 2)

2. **Currency Converter** (`Currency_Converter.dart`)
   - Konversi mata uang ke 6 jenis mata uang (IDR, USD, EUR, JPY, SGD, MYR)
   - Menggunakan kurs yang realistis

2. **Currency Converter** (`Currency_Rupiah_Converter.dart`)
   - Konversi mata uang rupiah ke 4 jenis mata uang (IDR, USD, EUR, JPY, KWR)
   - Menggunakan kurs yang realistis

3. **All Operators Demo** (`All_Operators.dart`)
   - Demonstrasi semua jenis operator Dart:
     * Operator Aritmatika
     * Increment & Decrement
     * Perbandingan / Relasional
     * Logika
     * Assignment
     * Bitwise
     * Ternary
     * Type Test
     * Null-aware
     * Cascade

### II. TANTANGAN TAMBAHAN

4. **Unit Converter App** (`Unit_Converter.dart`)
   - Konversi 4 kategori: Panjang, Massa, Volume, Suhu
   - Minimal 5 unit per kategori
   - Validasi input (hindari nilai negatif)
   - Format hasil yang rapi
   - Implementasi Map untuk faktor konversi
